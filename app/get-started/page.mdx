## Prerequisites

Before you can start deploying you should have the following secrets:

- CLOUDFLARE_API_TOKEN
- CLOUDFLARE_DEFAULT_ACCOUNT_ID
- CLOUDFLARE_EMAIL
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
